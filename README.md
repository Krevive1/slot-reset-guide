# slot-reset-guide

初心者向けの「パチスロ朝一リセット狙い」情報サイトです。

## サイト概要

このサイトでは、パチスロ初心者向けに以下の内容を解説します。

- 朝一リセットとは何か
- リセット恩恵とは何か
- 初心者が注意すべきポイント
- 過度な投資を避けるための考え方

## ファイル構成

- `index.html`：トップページ
- `beginner.html`：初心者向け解説ページ
- `disclaimer.html`：免責事項
- `privacy.html`：プライバシーポリシー
- `styles.css`：共通スタイル

## GitHub Pages 公開手順

1. GitHubのリポジトリで `Settings` を開く
2. 左メニューから `Pages` を開く
3. Sourceを `Deploy from a branch` にする
4. Branchを `main` にする
5. Folderを `/ root` にする
6. Saveを押す
